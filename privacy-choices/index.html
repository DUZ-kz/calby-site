In-app controls available today

  • Edit preferences: update profile details and goals inside the app.
  • Reminder settings: adjust notification preferences from the Profile screen.
  • Reset my data: clear synced profile and app-state data for the signed-in account and remove related local data on the current iPhone.
  • Delete account: permanently remove the Calby account, synced data, and local files tied to that account on the current iPhone.
  • Sign out: remove the current session from the device.

Requests that should go through support

  • Account deletion follow-up requests
  • Questions about synced data retention
  • Questions about AI scan processing
  • Requests related to privacy laws that apply in your region

How to request help: email calby.support@gmail.com or use the hosted support page.

Current data model note

Profile information, synced app state, and backed-up body progress photos can be restored after sign-in on a new phone.
